# 情報設計変更メモ
**作成日**: 2026-03-20
**対象**: Pairlog v1（2026-03-20 再構築）

---

## タブ構成の変更

### 旧（v0）タブ: 5タブ
| タブ名 | パス | 役割 |
|---|---|---|
| ホーム | `(home)/index` | 週次サマリー |
| 記録 | `(home)/events` | 記録入力 |
| 約束 | `(home)/promises` (stack screen) | 約束一覧 + カレンダー埋め込み |
| 解放 | `(home)/repair` | 解放手続き |
| 設定 | `(home)/settings` | 設定 |

問題点: 週次サマリーが先頭 / カレンダーが約束タブの奥に埋まっている / 解放タブが独立しすぎ

### 新（v1）タブ: 4タブ
| タブ名 | パス | 役割 | アイコン |
|---|---|---|---|
| ホーム | `(home)/index` | 今日の約束ダッシュボード | home |
| 約束 | `(home)/promises` | 約束管理 + インライン記録 | checkmark-circle |
| カレンダー | `(home)/calendar` | 月次カレンダー + 日別約束 | calendar |
| 設定 | `(home)/settings` | 設定・ペア管理 | settings |

### 非表示ルート（タブバーから消えたが routes は維持）
- `(home)/events` — 旧・記録タブ（ホームから push で到達可能）
- `(home)/repair` — 解放手続き（ホームの解放カードから push で到達）
- `(home)/summary` — 週次サマリー（ホーム下部のリンクから push で到達）

---

## ホーム画面の変更

### 旧ホーム
- ファーストビュー: 週次ポイント集計
- 今週の約束リスト（全件）
- 解放への誘導

### 新ホーム
- ファーストビュー: **今日の約束**（`isRuleActiveToday()` フィルタ済み）
- 各約束に「記録」ボタン → インラインで `createEvent()` 実行
- カレンダーへのショートカットカード
- 今週のポイント（サブ表示）
- 解放カード（open repair がある場合のみ表示）
- 週次サマリーへの補助リンク

---

## 約束タブの変更

### 旧（/promises — stack screen）
- カレンダー埋め込み（tab で切り替え）
- ルール一覧
- 記録は events screen へ push

### 新（(home)/promises — bottom tab）
- フィルタ: すべて / ルーティン / スポット
- 各カードに記録ボタン → インライン記録
- カレンダーへの導線なし（専用タブが独立）
- 空状態: 8種テンプレートサジェスト
  - 連絡・共有 / 感謝 / 子育て / 家事分担 / ふりかえり

---

## カレンダータブ（新設）

- 月次カレンダー ← 旧は promises タブ内に埋まっていた
- ドット表示: オレンジ=ルーティン、ブランドピンク=スポット
- 日タップ: その日の約束リスト + インライン記録ボタン
- 今日ボタン: 常に今日へ戻れる

---

## create-rule.tsx の変更

- 保存後ナビゲーション: `router.replace('/promises?created=...')` → `router.replace('/(home)/promises')`
- RULE_SUGGESTIONS 拡張: 4種 → 17種（7カテゴリ対応）
