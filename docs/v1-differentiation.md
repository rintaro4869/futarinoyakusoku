## 競合ポジショニング

```
            高機能・多機能
                 ↑
         TimeTree │ Notion / タスク管理
                  │
家族向け ─────────┼───────── カップル向け
                  │
     Pairlog v1  │ Paired
  (約束OS / 実用) │ (エンタメ・Q&A)
                  │
                 ↓
            シンプル・日常使い
```

**Pairlog のポジション**: 「家族向け × シンプル × 日常使い」

## Pairlog が勝てるポイント

### 1. 「約束」というフレーミング
家事を「やるべきタスク」でなく「ふたりで合意したこと」として捉え直す。
「なんでやらないの」ではなく「この約束どうしよっか」という会話を促す。

### 2. スポット記録（気づいたとき型）
定期的なルーティン以外に「その場で嬉しかった行動」を記録できる。
TimeTree・CAJICO にはない「感謝の蓄積」機能。

### 3. ふりかえりサポート
週次・月次の振り返りをアプリが促す（テンプレート + 週次サマリー）。
これにより「ふたりで話し合う習慣」が形成される。

### 4. 子育て家庭に刺さるテンプレ
「保育園のお迎え担当」「翌日の持ち物確認」など、
既存競合がカバーできていない具体的な家庭運営シーンを提供。

### 5. 安全機能の存在
DV 被害のリスクを想定した「ペアリング解除」「アカウント削除」への
常時アクセシブルな経路。競合アプリにはほぼ存在しない。

---

## 勝てないポイント（正直に）

- **カレンダー共有の深さ**: TimeTree には勝てない。Googleカレンダーとの同期等は対象外。
- **ゲーミフィケーションの楽しさ**: CAJICO のポイント交換・ランキングには勝てない。
- **コンテンツの豊富さ**: Paired の Q&A・デートアイデアには勝てない。

→ これらは対象外として明確に割り切る。

## マーケティングコピー案

**メインコピー**
「ふたりで決めたことを、ふたりで守る。」

**サブコピー**
「家事を「誰かのせい」にしない。約束を可視化すれば、感謝が生まれる。」

**キャッチフレーズ**
「共働き家庭の約束OS」

**ストアカテゴリ**: ライフスタイル（サブ: 生産性）
