# QA チェックリスト（TestFlight 版）
**対象**: ペアログ v1.0
**目的**: 致命的なバグがないことを確認する。全20項目。

---

## 基本フロー（1台で確認）

- [ ] **1. 起動** — アプリをタップすると起動する（クラッシュしない）
- [ ] **2. 新規登録** — メール・パスワードで新しいアカウントを作れる
- [ ] **3. チュートリアル** — スライドをスワイプして「始める」まで進める
- [ ] **4. ログアウト→再ログイン** — 設定からログアウトし、同じ情報で再ログインできる
- [ ] **5. ログイン維持** — アプリを完全に閉じて再起動してもログイン状態が保たれる
- [ ] **6. 間違ったパスワード** — 誤パスワードでログインするとエラーメッセージが出る（クラッシュしない）

---

## ペアリング（2台で確認）

- [ ] **7. 招待コード作成** — ペア設定画面で招待コードが表示される
- [ ] **8. 招待コードで参加** — 端末Bで招待コードを入力してペアリングできる
- [ ] **9. ペアリング解除** — 設定から解除でき、ペア設定画面に戻る

---

## 約束・記録（コア機能）

- [ ] **10. 約束を作る** — タイトル・種別・ポイントを設定して保存できる
- [ ] **11. 一覧に表示** — 作った約束が約束タブに表示される
- [ ] **12. 記録する（ホーム）** — ホームの「記録する」ボタンで記録でき、ポイントが増える
- [ ] **13. 記録する（約束タブ）** — 約束タブからも記録でき、ポイントが増える
- [ ] **14. 2台連動** — 端末Aで記録すると端末Bのポイントも増える
- [ ] **15. 約束を削除** — 削除すると一覧から消える（確認ダイアログあり）

---

## カレンダー・設定

- [ ] **16. カレンダー** — 記録した日にドットが表示される
- [ ] **17. 月移動** — 「<」「>」で前月・翌月に移動できる

---

## 安全・異常系

- [ ] **18. 通知を拒否してもアプリが使える** — 通知を「許可しない」にしてもクラッシュしない
- [ ] **19. 機内モード** — 機内モードにしてもアプリがクラッシュしない（エラー表示が出る）
- [ ] **20. アカウント削除** — 設定から削除でき、その後ログインできなくなる

---

## 判断基準

| 🔴 即修正 | アプリが止まる・操作できない・データが消える |
|----------|--------------------------------------|
| 🟡 提出前に修正 | ボタンが反応しない・画面が崩れる |
| 🟢 後でOK | 余白のズレ・文言の微調整 |
