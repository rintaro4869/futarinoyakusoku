## 説明文（最大 4,000 文字）

```
■ Pairlog は、カップルのための「記録」アプリです

「なんで言ってくれないの」
「また守れなかった」

そんな言葉のかわりに、ふたりで決めた約束を静かに記録していく。
それが Pairlog のコンセプトです。

──────────────────────
▶ 基本のながれ
──────────────────────

1. ふたりで「約束」を作る
   例:「連絡は当日中に返す」「週に 1 回、ゴミを出す」
   あらかじめ決める「ルーティン」と、
   その場で記録する「スポット（思いがけないありがとう）」の 2 種類。

2. 毎日、できたら記録する
   「約束できた」または「思いがけないありがとう」のボタンを 1 回タップ。
   非難も採点もありません。ただ記録するだけ。

3. ポイントがたまったら「解放」
   サンキューポイントと約束ポイントが上限に達すると、
   「解放」アクションを実行できます。
   ごほうびでも、話し合いのきっかけでも——中身はふたりで自由に決めてください。

4. 週に一度、ふりかえる
   週次サマリーで「どの約束がうまくいったか」を確認。
   責めるためではなく、整えるための記録です。

──────────────────────
▶ 安心して使える設計
──────────────────────

・メールアドレス不要で始められます
・いつでもアプリを「一時停止」できます
・パートナーとの繋がりを解除することもできます
・データ削除はアプリ内のメニューからいつでも可能です

──────────────────────
▶ こんな方に
──────────────────────

・「言った言わない」で喧嘩になりがちなカップル
・同棲中で家事や連絡の分担を整えたい
・パートナーとの関係をよくしたいけど、会話のきっかけが難しい
・責め合うのではなく、ふたりで一緒に良くなりたい

──────────────────────
▶ 注意事項
──────────────────────

本アプリは監視・管理ツールではありません。
ふたりの合意のもとで使うことを前提に設計されています。
不安や恐怖を感じさせる使い方には対応していません。
```

## 審査担当者へのメモ（App Review Notes）

> App Store Connect の「審査に提出」→「App Review 情報」→「メモ」欄に貼ります。

```
Dear App Review Team,

Thank you for reviewing Pairlog.

【 What this app does 】
Pairlog is a promise-tracking app for couples.
Two users (partners) create shared "rules" (promises), record when they keep them,
and earn points together. When points reach the threshold, they can trigger
a "release" action — such as a reward or a conversation starter — that
they define themselves.

【 How to test with two accounts 】
The app requires two separate devices (or two Apple accounts in TestFlight).

Step 1 — User A:
  1. Open the app
  2. Tap "パートナーなしで始める" (Start without partner)
  3. Enter a display name → Tap "作る" (Create)
  4. You will see an invite code on screen (e.g. "ABC-123")

Step 2 — User B:
  1. Open the app on a second device
  2. Tap "招待コードで参加する" (Join with invite code)
  3. Enter the code from User A → Tap "参加する" (Join)

After pairing, both users can create rules, record events, and see the home screen.

【 Demo account (single device) 】
If testing on a single device is preferred:
- Create one account as User A (with display name "レビュアーA")
- The invite code can be entered manually on the same device
  by reinstalling the app and signing in as a new user.
- Alternatively, please contact us for a test invite code at: rintaro4869@gmail.com

【 No email or sign-up required 】
Authentication is fully anonymous. No email address is needed.

【 Safety features 】
- Settings screen → "アプリを一時停止" pauses the couple's activity
- Settings screen → "パートナーとの接続を解除" leaves the couple
- Settings screen → "データを削除する" deletes all user data

Thank you for your time.
```
