# TestFlight 確認チェックリスト

> TestFlight ビルドが届いたら、このリストを 2 台の端末で実施してください。
> 全項目 OK になったら App Store 提出できます。

---

## 前準備

- [ ] TestFlight アプリ（Apple 公式）をインストール済み
- [ ] 招待メールから TestFlight でインストール完了
- [ ] 2 台の iPhone（または 1 台 + シミュレーター）を用意

---

## A. 初回起動フロー

- [ ] アプリを開くとチュートリアル（使い方ガイド）が表示される
- [ ] 4 ステップをすべてスワイプできる
- [ ] 「始める」を押すとホーム画面 or ペア設定画面に進む
- [ ] 表示が崩れていない（文字が見切れていない）

---

## B. ペア接続

**端末 A（ホスト側）**

- [ ] 「パートナーなしで始める」をタップ
- [ ] 表示名（例: 「自分」）を入力して「作る」をタップ
- [ ] 招待コード（例: `ABC-1234`）が画面に表示される
- [ ] コードをコピーできる
- [ ] 「共有する」ボタンが動作する

**端末 B（参加側）**

- [ ] 「招待コードで参加する」をタップ
- [ ] 表示名を入力して「参加する」をタップ
- [ ] ホーム画面に遷移する

**接続後の確認**

- [ ] 両端末でホーム画面が表示される
- [ ] ポイントメーターが 0pt から始まっている

---

## C. 約束の作成

- [ ] 「＋ 約束を追加」または「最初の約束を作る」をタップ
- [ ] ルーティン（事前に決めた習慣）を 1 つ作成できる
- [ ] スポット（思いがけないありがとう）を 1 つ作成できる
- [ ] タイトルが一覧に表示される
- [ ] 3 つ目まで作成できる
- [ ] 4 つ目を作ろうとすると「上限」のメッセージが表示される（上限 3 件）

---

## D. ホーム画面

- [ ] サンキューポイントのメーターが表示される
- [ ] 約束ポイントのメーターが表示される
- [ ] メーターのラベルに説明テキスト（「ありがとうを形にしたポイント」など）が表示される
- [ ] 「今日の記録をする」ボタンが動作する
- [ ] ポイントが 0/0 のときに NaN や undefined が表示されない

---

## E. 記録画面（今日の記録）

- [ ] ルーティンのカードがオレンジのボタンで表示される
- [ ] スポットのカードがピンクのボタンで表示される
- [ ] ボタンをタップすると記録が完了し、成功メッセージが表示される
- [ ] 連続タップしても二重記録されない（3 秒以内の連打は無視）
- [ ] 記録後、ホームに戻るとポイントが増えている
- [ ] ルールを長押し or ••• メニューから削除できる

---

## F. 解放タブ（約束ポイント解放）

- [ ] テンプレートの一覧が表示される
- [ ] テンプレートをタップするとタスクに追加できる
- [ ] 追加後、「追加済み」の表示になる
- [ ] タスクを完了できる

---

## G. サマリー（週次ふりかえり）

- [ ] 今週の記録率が表示される
- [ ] 「記録できた割合」という表現になっている（「承認率」ではない）
- [ ] グラフが正しく表示される（0% / 100% が極端でも崩れない）
- [ ] 前週・次週のナビゲーションが動作する
- [ ] 未来の週には「まだデータがありません」など適切に表示される

---

## H. 通知

- [ ] 初回起動時に通知許可ダイアログが表示される
- [ ] 「許可しない」を選択してもアプリがクラッシュしない
- [ ] 「許可しない」を選択した後も通常通り使用できる
- [ ] 設定画面でリマインダーを有効にするとダイアログが再表示される（許可済みの場合は即時有効）

---

## I. 安全機能（セーフティ）

- [ ] 設定 → 「アプリを一時停止」が動作する
- [ ] 一時停止中はホーム画面に停止中の表示が出る
- [ ] 設定 → 「パートナーとの接続を解除」が動作する（確認ダイアログあり）
- [ ] 解除後、初期画面（ペア設定）に戻る
- [ ] 設定 → 「データを削除する」をタップすると確認ダイアログが表示される
- [ ] 確認後、データが削除されてアプリが初期化される

---

## J. エラー・オフライン

- [ ] 機内モードにしてアプリを開くとエラーメッセージが表示される（クラッシュしない）
- [ ] 機内モードを解除してリフレッシュすると正常に表示される
- [ ] 約束が 0 件のときに空のメッセージ（「約束がまだありません」）が表示される

---

## K. UI の見た目チェック

- [ ] iPhone SE（4.7"）で見切れがない
- [ ] iPhone 15 Pro Max（6.7"）で余白が不自然でない
- [ ] ダークモードは想定外（Light モード固定なので OK）
- [ ] 日本語フォントが正しく表示される

---

## 合格基準

上記チェックリストで **赤字（クラッシュ・操作不能）** がなければ提出 OK です。
軽微な表示ズレは提出後も修正できます（アップデートで対応可能）。
