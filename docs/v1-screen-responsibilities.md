# 画面責務一覧
**作成日**: 2026-03-20
**対象**: Pairlog v1（4タブ構成）

---

## タブ画面

### `(home)/index.tsx` — ホームタブ
**責務**: 今日の約束ダッシュボード
**表示内容**:
- 今日の日付 + 「今日の約束」ヘッダー
- 今日有効な約束リスト（`isRuleActiveToday()` フィルタ）
- 各約束のインライン記録ボタン（debounce 3s、haptics）
- カレンダーショートカットカード
- 今週のポイント・MeterBar（ポイントがない場合は非表示）
- 解放カード（open repairs がある場合のみ）
- 週次サマリーへの補助リンク
- 初期状態（ルール0件）→ オンボーディング誘導
- 今日の約束なし → スポット作成誘導

**しないこと**: 約束の編集・削除・詳細表示

---

### `(home)/promises.tsx` — 約束タブ
**責務**: 約束管理と記録の統合ビュー
**表示内容**:
- フィルタセグメント（すべて / ルーティン / スポット）
- 全約束のカード一覧
- 各カードにインライン記録ボタン
- 空状態: 8種テンプレートカード（カテゴリ別）
- フィルタ不一致状態: 「作成する」ボタン

**しないこと**: カレンダー表示、約束の作成フォーム（`/create-rule` へ委譲）

---

### `(home)/calendar.tsx` — カレンダータブ
**責務**: 月次カレンダーと日別約束の確認・記録
**表示内容**:
- 月次カレンダー（前後月ナビ、今日ボタン）
- 各日のドット表示（ルーティン=オレンジ、スポット=ピンク）
- 日タップ → 下部に日別約束パネル
- 日別パネル: 各約束にインライン記録ボタン
- 空状態（ルール0件）→ 「約束を作る」ボタン

**しないこと**: 週表示、イベントの詳細編集

---

### `(home)/settings.tsx` — 設定タブ
**責務**: ユーザー設定・ペア管理・安全機能
**表示内容**:
- パートナーとのペアリング情報
- ペアリング解除（常時 accessible）
- アカウント削除（常時 accessible）
- ログアウト
- アプリバージョン表示

**しないこと**: 約束の設定（`/create-rule` 担当）

---

## モーダル・スタック画面

### `/create-rule.tsx`
**責務**: 新規約束の作成フォーム
**入力**: `?title=` `?mode=` のクエリパラメータ（テンプレートから遷移時）
**完了後**: `/(home)/promises` へ replace ナビゲーション

### `/edit-rule.tsx`
**責務**: 既存約束の編集
**入力**: `?id=` クエリパラメータ
**完了後**: `router.back()` で呼び出し元に戻る

### `(home)/events.tsx`
**責務**: 記録一覧 + 手動記録（旧・記録タブ）
**アクセス方法**: ホームから `router.push('/(home)/events')`
**タブバーには表示しない**（`options={{ href: null }}`）

### `(home)/repair.tsx`
**責務**: 解放（約束ポイント消化）手続き
**アクセス方法**: ホームの解放カードから `router.push('/(home)/repair')`
**タブバーには表示しない**

### `(home)/summary.tsx`
**責務**: 週次サマリー（ポイント詳細・振り返り）
**アクセス方法**: ホーム下部の補助リンクから push
**タブバーには表示しない**

### `/tutorial.tsx`
**責務**: 初回起動チュートリアル（4ステップ）
**アクセス方法**: 初回起動時に自動遷移

### `/login.tsx`・`/pair.tsx`・`/join.tsx`
**責務**: 認証・ペアリングフロー
**アクセス方法**: 未認証時に自動遷移

## データフロー

```
API (Cloudflare Workers + D1)
  │
  ├─ getHomeSummary(coupleId) → ホーム: ポイント・解放情報
  ├─ getRules(coupleId)       → ホーム・約束・カレンダー: 約束一覧
  ├─ createEvent(coupleId, ruleId) → 記録作成（ホーム・約束・カレンダーから）
  ├─ createRule(coupleId, ...)  → 約束作成（create-rule から）
  └─ deleteRule(coupleId, id)   → 約束削除（edit-rule から）

ローカル状態 (Zustand)
  └─ useAuthStore: coupleId, userId, token
```
